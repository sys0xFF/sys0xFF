### ⭐ Destaques

**[CVE-2025-61155](https://www.cve.org/CVERecord?id=CVE-2025-61155)** — pesquisador
co-creditado (com Gabriel Maciel Ramos e Gabriel Gomes).
Falha de controle de acesso em um driver de kernel assinado no Windows (`GameDriverX64.sys`):
um IOCTL acessível sem privilégios alcança `ZwTerminateProcess` em contexto de kernel,
permitindo encerrar processos arbitrários — inclusive serviços de segurança protegidos
(classe BYOVD / *EDR-killer*). NVD: CWE-400 · CVSS 5.5.
